# Pro Automobile – Zeiterfassung (PWA)

## 1) Supabase
- Tabellen + Policies per SQL anlegen (siehe Chat-Anleitung)
- In Vercel Env Vars setzen:
  - NEXT_PUBLIC_SUPABASE_URL
  - NEXT_PUBLIC_SUPABASE_ANON_KEY

## 2) Vercel
- Repo importieren
- Deploy
- Link auf Samsung in Chrome öffnen → ⋮ → "App installieren"

## Local (optional)
npm install
npm run dev
